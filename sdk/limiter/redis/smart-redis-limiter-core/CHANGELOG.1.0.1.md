# 1.0.1

## 新增

- `SmartRedisLimiterConstant` - 统一常量定义接口，含以下常量分组：
  - **来源标识**：`SOURCE_INTERCEPTOR`、`SOURCE_ASPECT`
  - **注解**：`ANNOTATION_CLASS_NAME`
  - **缓存 Key**：`CACHE_KEY_SEPARATOR`
  - **HTTP**：`HTTP_STATUS_TOO_MANY_REQUESTS`、`HTTP_MESSAGE_TOO_MANY_REQUESTS`、`HEADER_RETRY_AFTER`、`HEADER_X_FORWARDED_FOR`、`HEADER_X_REAL_IP`、`IP_UNKNOWN`
  - **时间单位后缀**：`SUFFIX_SECONDS`、`SUFFIX_MINUTES`、`SUFFIX_HOURS`、`SUFFIX_DAYS`
  - **异常消息**：`MSG_KEY_GENERATOR_NOT_FOUND`、`MSG_CLIENT_IP_NULL`、`MSG_REQUEST_PATH_NULL`、`MSG_METHOD_NULL`、`MSG_PATH_PATTERN_NULL`、`MSG_RATE_LIMIT_EXCEEDED`
