创建平台级AKSK

用于标识此AKSK的用途
多个权限用逗号分隔，支持通配符 (*)。不允许使用回车换行
⚠️ 检测到回车换行或空格，请使用逗号分隔
Scope 格式说明：
接口权限：如 /api/*（通配符 * 匹配所有路径），多个接口用逗号分隔，如 /api/orders,/api/products
数据权限：如 data:*（访问全部数据）、data:dept:华东（按部门限制数据范围）
生效规则：接口权限和数据权限均为 AKP 固有能力，签发时确定，运行时不可覆盖；两者同时满足才放行
示例：/api/orders,/api/products,data:dept:华东 表示可调 orders 和 products 接口，且只能操作华东数据
