使用说明

        

        

            
1. 获取AKSK

            
在AKSK管理页面创建平台级或用户级AKSK，创建成功后会显示Client ID和Client Secret（仅显示一次，请妥善保存）。


            
2. 换取Token

            
使用Client ID和Client Secret通过OAuth2 Client Credentials Grant流程换取Access Token。


            
3. 使用Token

            
在API请求的Authorization头中携带Token：

            
Authorization: Bearer {access_token}


            
4. Token管理

            
可在Token管理页面查看、搜索、删除Token。
