# Changelog 1.0.4

## 改进

### 页面交互优化
- 修复编辑权限范围页面（`/admin/{clientId}` 详情页）未限制回车输入的问题
  - 添加键盘事件监听，阻止在 scope 输入框中按回车键
  - 添加粘贴事件监听，自动将粘贴内容中的换行符替换为逗号
  - 添加前端验证，提交前检查是否包含换行符
  - 与创建 AKSK 页面保持一致的用户体验

## 技术细节

### 修改的模板
- `src/main/resources/templates/admin/detail.html`
  - 添加 `scopeError` 错误提示元素
  - 添加 `keydown` 事件监听器阻止回车输入
  - 添加 `paste` 事件监听器自动处理换行符
  - 添加前端验证逻辑

### 用户体验改进
- **阻止回车**：按下回车键时显示错误提示，2秒后自动消失
- **智能粘贴**：粘贴包含换行符的内容时，自动替换为逗号并显示成功提示
- **前端验证**：提交前检查是否包含换行符，避免后端验证失败
