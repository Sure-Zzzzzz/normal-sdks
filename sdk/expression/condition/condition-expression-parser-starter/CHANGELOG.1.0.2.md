# Changelog - v1.0.2

## 发布日期

2026-04-22

## 版本类型

**Minor Release** - 新增时间范围关键字，向后兼容

## 变更概述

在 grammar 和 `TimeRangeKeywords` 中补充 `最近` 系列时间范围关键字，与已有的 `近` 系列并列支持。

---

## 变更内容

### `TIME_RANGE` 词法规则新增关键字

在原有 `近X` 系列基础上，新增 `最近X` 系列和中文数字系列（如 `最近五分钟`、`最近一天`）。

**分钟级**：`最近5分钟`、`近五分钟`、`最近五分钟`、`最近10分钟`、`近十分钟`、`最近十分钟`、`最近15分钟`、`近十五分钟`、`最近十五分钟`、`最近30分钟`、`近三十分钟`、`最近三十分钟`

**小时级**：`最近1小时`、`近一小时`、`最近一小时`、`最近6小时`、`近六小时`、`最近六小时`、`最近12小时`、`近十二小时`、`最近十二小时`、`最近24小时`、`近二十四小时`、`最近二十四小时`

**天级**：`最近1天`、`近一天`、`最近一天`、`最近3天`、`近三天`、`最近三天`、`最近7天`、`近七天`、`最近七天`、`最近14天`、`最近十四天`、`最近30天`、`最近三十天`、`最近60天`、`最近六十天`、`最近90天`、`最近九十天`

**周级**：`最近1周`、`近一周`、`最近一周`、`最近2周`、`近两周`、`最近两周`、`近二周`、`最近二周`

**月级**：`最近1个月`、`近一个月`、`最近一个月`、`最近2个月`、`近两个月`、`最近两个月`、`最近3个月`、`最近三个月`、`最近6个月`、`最近六个月`

**年级**：`最近1年`、`近一年`、`最近一年`、`最近2年`、`近两年`、`最近两年`、`最近3年`、`近三年`、`最近三年`

> 注：`最近30天` 映射 `LAST_1_MONTH`，`最近60天` 映射 `LAST_2_MONTHS`，`最近90天` 映射 `LAST_3_MONTHS`（按月近似，非精确天数）。

---

## 向后兼容性

✅ **完全向后兼容**

- 原有 `近X` 系列关键字全部保留，行为不变
- 新增 `最近X` 系列为别名，映射到相同的 `TimeRange` 枚举值
- 已有表达式无需任何修改
