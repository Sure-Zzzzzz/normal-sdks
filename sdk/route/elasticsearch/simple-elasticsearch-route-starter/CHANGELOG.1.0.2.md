# [1.0.2]

## ⚡ 性能优化
- SpEL 表达式缓存：缓存编译后的 Expression 对象，支持运行时上下文动态求值
- Pattern 编译缓存：正则表达式编译结果缓存，避免重复编译开销
- 路由规则排序缓存：启动时缓存排序后的规则列表，避免每次请求重复排序

## 🔧 配置增强
- 新增 `urls` 字段：推荐使用，支持完整 URL 格式（如：`http://localhost:9200`）
- 保留 `hosts` 字段：向后兼容旧配置方式
- 完善配置验证：启动时进行 16 项配置检查，快速失败定位问题

## ✅ 测试完善
- 新增 SpEL 缓存功能测试
- 新增路由模式匹配器测试（8 个测试用例）
- 新增配置验证测试（20 个测试用例）

## 📝 技术细节
- 零依赖缓存实现：使用 JDK 内置 ConcurrentHashMap
- Java 8 兼容：所有代码兼容 Java 8+
- 线程安全：所有缓存实现均为线程安全
